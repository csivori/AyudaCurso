Compilado de Instrucciones Organización de Código CSS

Para que se utiliza CSS: Es el lenguaje que se utiliza para aplicar un estilo a un HTML. Describe como deben mostrarse los elementos de un HTML

Estructura de CSS: selector { propiedad: valor ;}

Selectores +Usados:

*{} Selecciona todos los elementos del HTML
tag{} Cualquier tipo de tag HTML (body, header, main, footer, p, h1..h6, etc.)
tag.clase{} Selecciona los tag HTML con la clase .clase
tag1,tag2,tagN{} Selecciona todos los elementos de tipo tag1, tag2 y tagN HTML
#id{} Selecciona el elemento con este id en el HTML
.clase{} Selecciona todos los elementos de esta clase en el HTML
.clase1.clase2{} Selecciona todos los elementos que tienen las 2 clases en el HTML
.clase1 .clase2{} Selecciona todos los elementos que tienen la clase2 descendiendo de la clase1 en el HTML
Mas Selectores...

Pseudo-Clases:Las pseudo-clases sirven para definir un estado específico de un elementio.

Pseudo-Clases +Usadas:

selector:focus{} Aplica el estilo al elemento cuando esta en foco
selector:link{} Aplica el estilo a un elemento cuando aún esta no-visitado
selector:visited{} Aplica el estilo a un elemento cuando ya ha sido visitado
selector:hover{} Aplica el estilo al elemento cuando el ratón pasa por encima
selector:nth-child(n){} Aplica el estilo al elemento N de tipo selector que es N hijo de su padre
Mas Selectores...

Ej.1: Colorear el fondo del Header de Rojo, Body de Azul, Footer de Verde y el elemento con id=MenuPrincipal de Amarillo

header{background-color : #FF0000;}
footer{background-color : #00FF00;}
body{background-color : #0000FF;}
#MenuPrincipal{background-color : #00FFFF;}

Ej.2: Colorear los elementos de tipo a según el estado del link:

/* Link No Visitado aún */
a:link{color : #FF0000;}
/* Link Ya Visitado */
a:visited{color : #00FF00;}
/* Mouse sobre el Link */
a:hover{color : #FF00FF;}
/* Link Seleccionado */
a:active{color : #0000FF;}
