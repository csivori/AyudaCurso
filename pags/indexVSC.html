Atajos de Visual Studio Code
Shift + Alt + F: Parado en el Código Fuente, lo Formatea
Alt + Click: Permite el posicionamiento múltiple para Agregar, Borrar o Seleccionar múltiples areas
Ctrl + D: Permite la selección múltiple de cadenas similares a la seleccionada en primer término para Agregar, Borrar o Seleccionar múltiples areas
Ctrl + B: Muestra u Oculta el Arbol del Proyecto para agrandar el área de trabajo
Lorem###: Agrega la cantidad ### indicada de palabras aleatorias simulando un párrafo
