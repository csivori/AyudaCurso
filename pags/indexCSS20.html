Compilado de Instrucciones CSS de Márgenes: Margin y Padding

Margin

Esta propiedad de Estilo se usa para crear un espacio alrededor de un elemento, antes que inicie el área con fondo (background) y cualquier borde (border) definido.

Los valores pueden ser: auto el navegador calcula el márgen / length especifica un márgen en px, pt, cm, etc. / % especifica un márgen en % del ancho del elemento contenedor / inherit especifica que el márgen debe heredarse del elemento padre

Se puede definir uno para un lado en particular margin-top: 10px (o margin-right, margin-bottom, margin-left ) o también se puede definir varios lados al mismo tiempo:
    margin: 10px los 4 lados
    margin: 10px 5px 10px para Superior e Inferior y 5px para Derecho e Izquierdo
    margin: 10px 5px 2px 10px para Superior, 5px para Derecho e Izquierdo y 2px para Inferior
    margin: 10px 5px 2px 3px 10px para Superior, 5px para Derecho, 2px para Inferior y 3px para Izquierdo

IMPORTANTE: Cuando se superponen margenes de un elemento con el siguiente, se utiliza el mayor de ambos.

Padding

Esta propiedad de Estilo se usa para crear un espacio dentro del borde y alrededor del Texto o Imagen.

Los valores pueden ser: length especifica un márgen en px, pt, cm, etc. % especifica un márgen interno en % del ancho del elemento contenedor inherit especifica que el márgen interno debe heredarse del elemento padre

Se puede definir un lado en particular padding-top: 10px (o padding-right, padding-bottom, padding-left ) o también se puede definir varios lados al mismo tiempo:
    padding: 10px los 4 lados
    padding: 10px 5px 10px para Superior e Inferior y 5px para Derecho e Izquierdo
    padding: 10px 5px 2px 10px para Superior, 5px para Derecho e Izquierdo y 2px para Inferior
    padding: 10px 5px 2px 3px 10px para Superior, 5px para Derecho, 2px para Inferior y 3px para Izquierdo
