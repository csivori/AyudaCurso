Compilado de Instrucciones CSS de Flex-box: Flex-Direction, Flex-Wrap y Flex-Flow

flex-direction:

  • row: Alinea elementos a la parte izquierda del contenedor
  • row-reverse: Alinea elementos a la parte derecha del contenedor
  • column: Distribuye horizontalmente los elementos dentro del contenedor (sin espacio a izquierda ni a derecha del grupo)
  • column-reverse: Distribuye horizontalmente los elementos dentro del contenedor (asignando un margen izq. y der. a cada elemento)
Mas sobre Flex-Direction...

flex-wrap:

  • no-wrap: Cada elemento se ajusta en una sola línea
  • wrap: Los elementos pasan a la próxima línea
  • wrap-reverse: Los elementos pasan a la próxima línea en reversa

flex-flow: <flex-direction> <flex-wrap>
