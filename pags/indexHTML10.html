Compilado de Instrucciones HTML de Texto

Tratamiento para el Texto

(sin Tag) Texto sin Formato
<p> Parrafo de Texto
<h1>..<h6> Títulos con distinto tamaño
<abbr> Explicación de una Abreviatura
<br> Salto de Línea
<hr> Línea Separadora Horizontal
<!-- --> Texto para Comentarios en HTML

Etiqueta <font> para estilos del Texto (No mas soportada desde HTML5)

<p>Texto <font color="red">Rojo</p> <p>Texto <span style="color:red">Rojo</span></p>
mas nombres de color... mas colores en HEX...
<p>Texto <font size="8px">Chico</p> <p>Texto <span style="font-size:8px">Chico</span></p>

Formateo para el Texto

<b> Texto en negrita
<u> Texto subrayado
<i> Texto en itálica
<mark> Texto marcado
<ins> Texto insertado
<del Texto tachado
<small> Texto mas pequeño
<sup Texto en superindice
<sub> Texto en subindice
<em> Texto con énfasis
<strong> Texto importante

Ej.1: Uso de p, br, hr y abbr

Código HTML

<p>Parrafo 1: Este es el renglón 1<br />
El sujeto <abbr title="Carlos Alberto Sivori">CAS</abbr> tiene 53 años<br />
Y este es el renglón 3</p>
<p>Parrafo 2: Este es el renglón 1<br />Este el renglón 2</p> <hr />
<p>Parrafo 3: Este es el renglón 1<br />Este el renglón 2</p>

Salida

Parrafo 1: Este es el renglón 1
El sujeto CAS tiene 53 años
Y este es el renglón 3

Parrafo 2: Este es el renglón 1
Este el renglón 2

Parrafo 3: Este es el renglón 1
Este el renglón 2
