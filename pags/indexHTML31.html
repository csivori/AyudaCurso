Compilado de Instrucciones HTML de Listas con y sin Viñetas

<dl> Lista de Descripción (Identada)

<dt> Define el Item

<dd> Describe el Item

<ol> Lista Ordenada

<li> Item

<ul> Lista Desordenada

<li> Item

Ej.1: <ol> Lista Ordenada

con Viñeta Std

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

con Viñeta (start:5)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Viñeta Romana (type:I)

  1. Item 5
  2. Item 6
  3. Item 7
  4. Item 8

Viñeta Letras (type:a)

  1. Item 25
  2. Item 26
  3. Item 27
  4. Item 28

Ej.2: <ul> Lista Desordenada

con Viñeta Std

  • Item 1
  • Item 2
  • Item 3
  • Item 4

con Viñeta (type:circle)

  • Item 1
  • Item 2
  • Item 3
  • Item 4

con Viñeta (type:square)

  • Item 1
  • Item 2
  • Item 3
  • Item 4
